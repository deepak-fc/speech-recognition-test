Setting up the project

1. Download these files in the repo directory
		- https://alphacephei.com/vosk/models/vosk-model-small-en-us-0.15.zip
		- https://alphacephei.com/vosk/models/vosk-model-small-en-in-0.4.zip
		- https://alphacephei.com/vosk/models/vosk-model-en-in-0.4.zip

2. Extract zip as separate folder in the same directory

3. Install required libraries using pip
	- pip install pipwin
	- pipwin install pyaudio
	- pip install vosk

-------------------------------------------------------------------------------
How to Run?

1. Uncomment any one model line (Read main.py for that)
2. Run Main
